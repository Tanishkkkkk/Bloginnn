# bLOgINN System Architecture

This document outlines the core architecture, design decisions, and system modules of the bLOgINN application.

## 1. Technical Stack
- **Frontend Framework:** Next.js (App Router)
- **Styling:** Vanilla CSS with custom utility tokens
- **State Management:** Zustand
- **Database:** Prisma ORM with PostgreSQL

## 2. Component Design
Our component layer is designed to be highly modular and responsive. Theme states are synchronized locally and rendered seamlessly.
